# MyCalendar CLI

Удобная и быстрая консольная утилита для учета времени и ведения расписания прямо из терминала. Позволяет добавлять, редактировать, просматривать и удалять записи, а также экспортировать данные в CSV.

## Особенности
- **Кроссплатформенность:** Доступны сборки для Windows, macOS и Linux.
- **Работа без интернета:** Все данные хранятся локально в формате JSON.
- **Интерактивный UI:** Полноценное консольное меню с поддержкой цветного вывода.
- **Атомарные операции:** Безопасное сохранение данных и автоматические бэкапы.
- **Экспорт в CSV:** Удобная выгрузка данных для дальнейшего анализа в Excel или Google Таблицах.

## Запуск
Скачайте актуальную версию для вашей ОС из раздела [Releases](https://github.com/khameleonium/MyCal/releases) и запустите в терминале:

```bash
# Пример для Linux / macOS
chmod +x mycalendar-linux-amd64
./mycalendar-linux-amd64

# Пример для Windows
mycalendar-windows-amd64.exe
```

Или соберите из исходников:
```bash
go build -o mycalendar .
./mycalendar
```

## Быстрые команды
Утилиту можно запустить с флагами для быстрого выполнения действий без входа в главное меню:
- `mycalendar add` (или `a`) — быстрое добавление записей
- `mycalendar today` (или `t`) — показать записи на сегодня
- `mycalendar week` (или `w`) — показать записи за текущую неделю
- `mycalendar month` (или `m`) — показать записи за текущий месяц
